# TypeScript sorting

_Project for Udemy TypeScript course._

### About

Bubble sort example.

![drawing](https://github.com/ermondel/wiki/blob/master/files/drawings/drawing-ts-sort.png)

### Features

- Sort a collection of numbers
- Sort a collection of characters
- Sort a linked list

